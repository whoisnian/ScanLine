# ScanLine

### 环境
ArchLinux  
Breeze Dark Theme [KDE]  
Qt Creator 4.7.1 based on Qt 5.11.2  

### 操作
* 左键单击按钮进行绘图，查看五角星，三角形，六芒星，箭头示例。  
* 在绘图区域移动鼠标滚轮，对图像进行放大或缩小。  
* 在绘图区域按下鼠标左键进行拖动，调整图像显示位置。  
* 在绘图区域按下鼠标右键，重置绘图区域。
